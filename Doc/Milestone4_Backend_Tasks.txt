API 4: Delivery Failed
----------------------------------------------------------------

Endpoint      : PUT /api/shipper/orders/{orderId}/fail
Effort        : 3 hours
Priority      : High
Tables        : Orders, OrderItems, ProductVariants, Products, Payments, Notifications
Auth Required : Yes (Shipper)

Request Body:
{
  "reason": "Khach hang khong co nha"
}

Response Success (200):
{
  "success": true,
  "message": "Da ghi nhan giao hang that bai. Lan giao thu 2/3.",
  "data": { "deliveryAttempts": 2, "maxAttempts": 3 }
}

Response Error - Auto Cancel (200):
{
  "success": true,
  "message": "Giao hang that bai lan thu 3. Don hang tu dong huy.",
  "data": { "deliveryAttempts": 3, "status": "CANCELLED" }
}

Business Logic:
- Verify orderId co ShipperId = userId
- Check Status = 'SHIPPING'
- Tang DeliveryAttempts + 1
- Neu DeliveryAttempts >= 3:
  a. Status = 'CANCELLED', CancelReason = "Giao hang that bai 3 lan", CancelledAt = NOW
  b. Hoan tra StockQuantity, giam SoldCount
  c. Neu da thanh toan SePay: Payment.Status = 'REFUND_PENDING'
  d. Notification cho Customer: "Don hang bi huy do giao that bai 3 lan"
- Neu DeliveryAttempts < 3:
  Notification cho Customer: "Giao hang khong thanh cong. Se giao lai."

API 8: Get Chat Sessions
----------------------------------------------------------------
Task ID       : BE-4.2.2
Endpoint      : GET /api/chat/ai/sessions
Effort        : 2 hours
Priority      : High
Tables        : ChatAiHistory
Auth Required : Yes (Customer)

Response Success (200):
{
  "success": true,
  "data": [
    {
      "sessionId": "session-001",
      "lastMessage": "Ban muon xem chi tiet san pham nao?",
      "messageCount": 4,
      "createdAt": "2026-02-27T14:00:00Z",
      "updatedAt": "2026-02-27T14:30:00Z"
    },
    {
      "sessionId": "session-002",
      "lastMessage": "Minh goi y quan jogger big size ong rong...",
      "messageCount": 2,
      "createdAt": "2026-02-26T10:00:00Z",
      "updatedAt": "2026-02-26T10:15:00Z"
    }
  ]
}

Business Logic:
- Lay userId tu JWT
- Group ChatAiHistory theo SessionId
- Lay lastMessage = message cuoi cung cua moi session
- Count messageCount
- Sort theo updatedAt DESC

SQL Query Structure:
SELECT "SessionId",
       MAX("Content") FILTER (WHERE "CreatedAt" = max_time) as last_message,
       COUNT(*) as message_count,
       MIN("CreatedAt") as created_at,
       MAX("CreatedAt") as updated_at
FROM "ChatAiHistory"
WHERE "UserId" = @userId
GROUP BY "SessionId"
ORDER BY MAX("CreatedAt") DESC;
